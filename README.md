
## Overview
This repository contains a set of plugins designed to enhance the functionality of Sigil EPUB editor.
Each plugin addresses specific needs in EPUB processing and editing.

## Installation

To install these plugins in Sigil:
1. select the plugin folder you want to install
2. zip the folder
3. add plugin in sigil : Plugin -> Manage Plugins -> Add Plugin
4. run plugin in sigil : Plugin -> edit -> Choose the plugin

## Requirements
- Sigil EPUB Editor (version 2.7 or later recommended)
